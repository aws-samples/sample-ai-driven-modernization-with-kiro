# docs/

## How to Use

1. Copy the template files and fill them out:
   ```bash
   cp docs/project-info.template.md docs/project-info.md
   cp docs/modernization-requirements.template.md docs/modernization-requirements.md
   ```

2. Add `docs/project-info.md` to `.gitignore` (to protect sensitive information)

3. You can get started by filling out only the required fields. Optional fields will be collected by AI via Q&A.
